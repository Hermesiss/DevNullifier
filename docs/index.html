DevNullifier

Tired of 200 Gb AppData? Too tedious to delete Binaries manually from old Unreal Engine projects?

Features

🧹

Smart Cleaning

Automatically detects and cleans temporary files, caches, and junk data from your development environment.

📊

Size Visualization

See exactly how much space each folder is taking up with human-readable size formatting.

🎯

Selective Deletion

Choose exactly which folders to delete with checkboxes. Never delete something by accident.

⚙️

Configurable Scanning

Control scanning depth and customize patterns to match your development workflow.

📁

Multi-Platform Support

Works on Windows, Linux, and macOS. Cleans platform-specific cache and temp directories.

🛡️

Safe & Reliable

Only targets regenerable folders like node_modules, build artifacts, and cache directories.
